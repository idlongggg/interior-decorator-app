# 🚀 Interior Decorator API

Đây là phần xử lý AI (Backend) cho ứng dụng thiết kế nội thất.

## 🛠️ Cài đặt & Chạy ứng dụng

1.  **Cài đặt môi trường:**
    Khuyên dùng trình quản lý gói `uv`:
    ```bash
    uv sync
    ```

2.  **Cấu hình biến môi trường (`.env`):**
    ```env
    HF_API_KEY=hf_your_token_here
    ```

3.  **Khởi chạy API:**
    ```bash
    uv run python main.py
    ```

> Xem hướng dẫn chi tiết tại file `README.md` ở thư mục gốc của dự án.
