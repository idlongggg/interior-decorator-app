# 🎨 Interior Decorator App (Frontend)

Phần giao diện người dùng của ứng dụng thiết kế nội thất AI, được xây dựng bằng Next.js 15, React, và HeroUI.

## 🚀 Cách chạy ứng dụng

1.  **Cài đặt các gói:**
    ```bash
    pnpm install
    ```

2.  **Cấu hình API URL trong `.env`:**
    ```env
    NEXT_PUBLIC_API_URL=http://localhost:8000
    ```

3.  **Chạy server phát triển:**
    ```bash
    pnpm dev
    ```

Ứng dụng sẽ khả dụng tại [http://localhost:3000](http://localhost:3000).
